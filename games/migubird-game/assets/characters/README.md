## 通关合影

| 文件 | 用途 |
|------|------|
| `group.png` | 通关第 10 关后展示的咪咕鸟合影 |

建议尺寸：
- 角色图：64×64 或 128×128，透明背景
- 合影：横图约 800×500 或 1024×600，可包含三个角色

放入图片后刷新浏览器即可。未放置 `group.png` 时，通关页与主页会用三角色并排占位图代替。
